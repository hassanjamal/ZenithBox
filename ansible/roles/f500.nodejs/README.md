Nodejs
========

Install Nodejs and the NPM package manager

Requirements
------------

Debian Wheezy with the package python-pycurl and python-software-properties installed.

Example Playbook
-------------------------

    - hosts: servers
      roles:
         - { role: f500.nodejs }
